2. 什么是vue
是渐进式的JavaScript框架，采用 MVVM 模式
渐进式：vue不再去操作DOM元素来更新页面了，而是操作数据，页面会根据数据的变化自动更新。
       JQuery是DOM编程，vue是数据驱动编程。
MVVM：Model(数据模型 对象) 
      View(视图模型 DOM) 
      ViewModel(同步Model和View的对象 监听)

3.准备工作(安装插件)
浏览器插件：
Vue.js-devtools_v4.1.4  一个在浏览器上可以调试vue项目的插件

VS Code插件：
自动补全标签  
  Auto Close Tag
  Auto Rename Tag
  Auto Complete Tag
在 VS Code 的左侧菜单项最下面一个EXTENSIONS里搜索，比如 Auto Close Tag，然后 Install

开启一个服务器浏览HTML页面，可以实时更新页面
  Live Server
第一次使用需要 Command + Shift + p 输入 live server 选择 open

路径自动补全
  Path Intellisense

Vue语法高亮和自动补全代码
  Vetur
  VueHelper

VS Code配置：
打开 Code -> Preferences -> Settings -> Open Settings(JSON)
